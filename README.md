# AI Future Directions - Edge AI & AI-IoT Project

## Description
This repository demonstrates:
- Edge AI prototype (lightweight CNN for image classification)
- AI-IoT concept for smart agriculture

## Folder Structure

AI-Future-Directions/

├── data/

│ └── sample_images/

├── notebooks/

│ └── EdgeAI_Prototype.ipynb

├── src/

│ ├── train_model.py

│ ├── convert_to_tflite.py

│ └── test_edge_model.py

├── diagrams/

│ ├── edge_ai_workflow.txt

│ └── ai_iot_dataflow.txt

└── README.md


## Tools & Libraries
- Python 3.x
- TensorFlow / Keras
- TensorFlow Lite
- Jupyter Notebook
- NumPy, PIL

## Instructions
1. Train model: `python src/train_model.py`
2. Convert to TFLite: `python src/convert_to_tflite.py`
3. Test Edge AI model: `python src/test_edge_model.py`
4. Notebook demonstration: `notebooks/EdgeAI_Prototype.ipynb`

## Notes
- Sample images for testing should be placed in `data/sample_images/`.
- ASCII diagrams are in the `diagrams/` folder for reference.
